# HandMagic - Advanced Hand Tracking AR

HandMagic is a high-performance, web-based Augmented Reality (AR) experience that uses MediaPipe's hand tracking technology to create interactive visual and audio effects.

## Features
- **Real-time Hand Tracking**: Powered by MediaPipe Hands.
- **Dynamic Visuals**: Custom canvas-based particle systems and shockwaves.
- **Audio Feedback**: Procedural audio generated via the Web Audio API.
- **Multiple Themes**: Switch between Rainbow, Cyberpunk, Lava, Ocean, and Galaxy aesthetics.
- **Gesture Recognition**: Detects pinches and hand spread for interaction.

## Deployment
This project is deployed on GitHub Pages.

## Tech Stack
- HTML5 / CSS3 / JavaScript
- MediaPipe Hands
- Web Audio API
- Canvas API
